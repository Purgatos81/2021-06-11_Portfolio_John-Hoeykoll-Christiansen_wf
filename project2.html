SITE DEVELOPMENT

Planning

We had four weeks to complete this project, to design and create a Website, and the planning included creating a work schedule, including milestones for design, coding, testing and a report. I created a Gantt chart, and used it to keep track of my work hours. I also created a Trelloboard and used it both for planning and to take notes for later use in the report. I then went trough all the assets given to us in the assignment, and started planning by drawing a paper prototype. After that I created a prototype with Adobe xd.

Building

I started building this site for mobile devices first, and then expanded into tablet and desktop. It was a pretty straightforward process, adding navigation first and then the different pages etc. We did not implement JavaScript in this assignment. Finally it was bugtesting, validation and tweeking.

Evaluation

Looking back at this project, I think what this page needs more is whitespace. The footer also needs some work, and the contact form must be written with JavaScript and redesigned. The search function can also be implemented. I would also like to work on the navigation tab description.
