Nordic Light Hotels Website

A Hotel booking website for a local agency called Northern Light Hotels, where visitors can find accommodations, book hotels and contact the owners. The admin can also add hotels, and view and manage both enquiries and messages from customers. The hotels and enquiries are stored on a Strapi API hosted on Heroku

View Website View GitHub

SITE DEVELOPMENT

Planning

We had 8 weeks for this project. I started planning by creating a Gannt chart, to help me manage the process. After that, i went online for inspiration, downloading images and made a Mood board. Lastly I made a Figma prototype to help me visualise the page.

Building

This page was built using React.js and styled components. I started with some components, like navigation and footer, and made sure they where responsive. From there I created the homepage, added the API calls and JavaScript. When most of the code was working, I made the rest of the site.

Evaluation

If I return to this project later, my plans are to add more items to the front page, and more options to the admin page, like deleting messages and hotels, and the option to edit them.
