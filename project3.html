Design Speakers

A web page for ordering speakers online, and for the admin to add, uppdate and delete products. The users can add and remove products from the shopping cart, search for products and view detailed information about the speakers. The admin can log in, uppdate, add and delete products. Users are also prevented from accessing the admin page unless they have logged in.

View Website View GitHub

SITE DEVELOPMENT

Planning

This project was planned using Gantt chart, and Figma for prototype. I used other e-commerce sites for inspiration, and used a local strapi setup for the data and API calls. All images comes from Adobe stock.

Building

This site was built with basic HTML, css and javascript, using Sass for styling the CSS. The logo was created using Adobe XD.

Evaluation

This was the first project with more heavy use of Javascript. There where login features, add to/remove from chart and add/ uppdate options to administration page. If I where to work on this project again, I would add more items to the page, create an online database, and add more options for the admin section.
