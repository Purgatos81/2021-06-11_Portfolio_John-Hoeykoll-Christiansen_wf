SITE DEVELOPMENT

Planning

Since this project was a cross-course assignment, some parts of the planning was done before the week of production. Most of the top-navigation was made beforehand, and gave me a quick start. With only a week to produce the site, the prototype was created using Adobe XD, and from there it went straight to koding. I created a repo with GitHub and downloaded it to my local desktop and VSCode.

Building

Since this site was built for desktop only, I started with the navigation, and created all the pages needed. Then I had to work with images and Photoshop, before putting it all together with VSCode. Since we had not learned Javascript yet, input fields where non functional, and just for visual presentation.

Evaluation

Being the first project, there aer a lot of improvements to be made to this site. First, I am not happy with the colors, and I would do a lot of work with finding new colors, gradients, shadows etc. The fonts also need an update, and the pages need more whitespace. The top navigation also needs some tweaking. The CTA button needs to stand more out. I would also need to do more work on the game specific page.
