TWC-Blog

This was the final project of my first year at Noroff. In this assignment, I created a Blog for TWC-Media using WordPress to retrieve a headless CMS for the content. It was also my first time using JavaScript.

SITE DEVELOPMENT

Planning

The first thing I did in this assignment was figuring out what to make in this project. We had to design a blog page, but we could decide for ourselves what type of blog it was. I spent some time on the internet looking for ideas, and found out that I wanted something that interested me. I contacted TWC-Media and asked for permission to use their photoes for my project, and luckily for me they agreed. After I had an idea for the project, I started setting up a Gantt chart for time management, and started sorting what I needed. I spent a lot of time talking with TWC-Media, and worked on the pictures and stories they sent me.Then I created a prototype in Figma.

Building

To build this site, I used GitHub and VScode, and deployed it to Netlify. I used Wordpress to retrieve a headless CMS for the content in the blogs. This time I used JavaScript to make API calls and validate form input.

Evaluation

My next plans for this site is to evaluate it after it is graded, but I also am looking to putting in a search option, and I might also change the fonts.
